### Development

Want to contribute? Great!

Open 3 tabs of your favorite Terminal and run these commands.

First Tab:
```sh
$ cd client && npm i && npm start
```

Second Tab:
```sh
$ cd scrapper && pip install requirements.txt && cd app && python main.py
```

Third Tab:
```sh
$ cd server && npm i && npm run dev
```
